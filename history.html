Databases for Many Majors

2020

  • Java version that includes a choice of customization example, distributed with the Java runtime for executing the visualizations.

2019

  • Open Science Framework - Databases for Many Majors, https://osf.io/qgepn/, Project's archival site with an overview of each visualization including screen shots and the source code for the FLEX versions.
  • Java versions of the default visualizations (students taking courses) are now available! Customized Java versions to follow soon. Just download the jar file to your local machine with a JRE (Java Runtime Environment) and open the jar with a double-click. NOTE: Required Java 8
  • J. E. Broatch, S. W. Dietrich and D. Goelman, Introducing Data Science Techniques by Connecting Database Concepts and dplyr, Journal of Statistics Education, September 2019, https://doi.org/10.1080/10691898.2019.1647768.
  • Neuroinformatics Customizations available for IntroDB, QueryDB and DesignDB!
  • R. Bingham, S. W. Dietrich and D. Goelman, Strategies to improve accessibility for learners with color vision deficiency, ACM Inroads: Volume 10 Issue 1, March 2019, pp. 52-56, https://doi.org/10.1145/3306140.

2018

  • Instructors page updated to include a cooperative exercise for Conceptual Database Design
  • Brighter colors introduced as part of a reflection on the accessibility of the visualizations for those with color deficient vision.
    Thanks to the following tools for providing assistance: Cynthia A. Brewer's Colorbrewer and WebAIM's Contrast Checker.
