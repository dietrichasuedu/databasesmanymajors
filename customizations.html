Customizations


The visualizations have a choice of STEM examples thanks to our collaborators acknowledged below. Each data customization has an Access database and scripts for creating and populating MySQL databases. Additionally, some collaborators have included links to resources, more detailed data descriptions, and cooperative learning exercises. The last data example in the list is the default Students Taking Courses example, which also has a companion document illustrating how to query in Access and MySQL.
